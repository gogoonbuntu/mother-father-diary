# 데이터 보안 정책

## 수집 및 공유되는 데이터

럭키비키 일기장 앱은 사용자 경험을 향상시키고 서비스를 제공하기 위해 다음과 같은 데이터를 수집합니다:

### 수집하는 데이터

1. **개인 정보**
   - 이메일 주소 (Google 로그인을 통해)
   - 이름 (Google 로그인을 통해)

2. **사용자 생성 콘텐츠**
   - 일기 내용
   - 기분/감정 상태
   - 일기 작성 날짜
   - 긍정적 버전 텍스트

3. **앱 활동 정보**
   - 앱 사용 빈도
   - 기능 사용 패턴
   - 앱 성능 데이터
   - 오류 로그

4. **기기 정보**
   - 기기 모델
   - 운영 체제 버전
   - 앱 버전
   - 언어 설정

### 데이터 사용 목적

수집된 데이터는 다음과 같은 목적으로 사용됩니다:

1. **앱 기능 제공**
   - 사용자 인증 및 계정 관리
   - 일기 데이터 저장 및 동기화
   - 긍정적 버전 생성 (Gemini API 사용)

2. **앱 개선**
   - 사용자 경험 향상
   - 버그 수정 및 성능 최적화
   - 새로운 기능 개발

3. **분석**
   - 앱 사용 패턴 이해
   - 기능 인기도 측정
   - 성능 모니터링

### 데이터 공유

1. **제3자 서비스 제공업체**
   - Firebase (사용자 인증, 데이터 저장)
   - Google Cloud (Gemini API)
   - Google AdMob (광고 표시)

2. **데이터 공유 방식**
   - 모든 데이터는 암호화되어 전송
   - 제3자 서비스 제공업체는 자체 개인정보처리방침에 따라 데이터 처리
   - 사용자의 명시적 동의 없이 마케팅 목적으로 데이터 공유 안 함

## 데이터 보안 조치

럭키비키 일기장 앱은 사용자 데이터를 보호하기 위해 다음과 같은 보안 조치를 구현합니다:

1. **데이터 암호화**
   - 전송 중 데이터 암호화 (HTTPS/SSL)
   - 저장 데이터 암호화 (Firebase 보안 규칙)

2. **접근 제어**
   - 사용자 인증 필수
   - 각 사용자는 자신의 데이터에만 접근 가능

3. **정기적인 보안 검토**
   - 코드 보안 검토
   - 취약점 테스트
   - 보안 패치 적용

4. **데이터 최소화**
   - 필요한 최소한의 데이터만 수집
   - 불필요한 데이터는 수집하지 않음

## 데이터 삭제 및 사용자 권리

1. **데이터 삭제**
   - 사용자는 앱 내에서 계정 삭제 가능
   - 계정 삭제 시 모든 개인 데이터 영구 삭제
   - 일기 항목 개별 삭제 가능

2. **데이터 접근 및 수정**
   - 사용자는 언제든지 자신의 데이터에 접근 가능
   - 데이터 수정 및 업데이트 가능
   - 데이터 내보내기 기능 제공

3. **옵트아웃 옵션**
   - 분석 데이터 수집 옵트아웃 가능
   - 광고 개인화 옵트아웃 가능
   - 알림 비활성화 옵션

## 어린이 데이터 보호

럭키비키 일기장 앱은 13세 미만의 어린이를 대상으로 하지 않으며, 고의로 13세 미만 어린이의 개인정보를 수집하지 않습니다. 13세 미만 사용자의 개인정보가 수집된 것이 확인되면 즉시 해당 정보를 삭제합니다.
