럭키비키 일기장 (럭키비키 일기장)

시행일: 2026년 4월 1일 | 최종 수정: 2026년 4월 26일

1. 수집하는 개인정보

럭키비키 일기장(이하 "앱")는 서비스 제공을 위해 최소한의 정보를 수집합니다:

  • Apple/Google 로그인 시: 이메일 주소, 이름 (선택사항)
  • 게스트 로그인 시: 별도의 개인정보를 수집하지 않습니다
  • 앱 사용 시: 사용자가 작성한 일기 내용

2. 개인정보의 이용 목적

  • 사용자 인증 및 계정 관리
  • 일기 데이터의 저장 및 동기화
  • AI 일기 변환 서비스 제공
  • 앱 사용 통계 분석 (비식별 데이터)

3. 개인정보의 보관 및 보호

모든 데이터는 Google Firebase에 암호화되어 안전하게 저장됩니다. 사용자의 일기 데이터는 본인만 접근할 수 있으며, 제3자에게 공유되지 않습니다.

4. 제3자 제공

앱은 다음의 제3자 서비스를 사용합니다:

  • Firebase (Google): 인증, 데이터 저장, 분석
  • Google AdMob: 광고 서비스 (IDFA 사용)
  • OpenAI API: AI 일기 변환 (일기 내용이 API로 전송되며, 저장되지 않습니다)
  • Microsoft Clarity: 앱 사용성 분석 (비식별 데이터)

5. 사용자 권리

사용자는 다음의 권리를 가집니다:

  • 데이터 삭제: 앱 설정에서 "계정 삭제" 기능을 통해 모든 데이터를 삭제할 수 있습니다
  • 데이터 접근: 앱 내에서 본인의 모든 일기 데이터를 열람할 수 있습니다
  • 동의 철회: 언제든지 앱을 삭제하여 서비스 이용을 중단할 수 있습니다

6. 아동 개인정보 보호

앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 개인정보 처리방침의 변경

본 방침이 변경될 경우, 앱 내 공지사항을 통해 사전에 안내합니다.
